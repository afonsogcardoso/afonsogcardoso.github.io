Project information

Category: Integration
Client: Production Agency

Overview

The integration aims to streamline the asset management and project workflow by providing seamless access to digital assets within the project management platform, facilitating efficient asset selection, request management, and task creation for asset-related work.

Core Functionalities

  • Embed DAM search functionality within the project management platform.
  • Allow users to filter and search for assets based on metadata, keywords, and other relevant criteria.
  • Enable the creation of assets selection into 'shopping carts' for batch management.
  • Provide an interface to request changes or modifications to selected assets.
  • Define a structured process for specifying asset changes (e.g., text adaptations, language updates, size requirements).
  • Automatically generate tasks based on asset requests and required modifications.
