Online Proofing

Category: Integration
Client: Production Agency

Overview

This integration leverages Ziflow's powerful proofing capabilities to streamline asset management and project workflow. By embedding Ziflow proofs directly within the platform via an iframe, users can efficiently review and approve creative assets. The integration allows for seamless proof creation through Ziflow's REST API, ensuring users can initiate proofing directly from the platform. Additionally, users can manage proof review stages by assigning participants and creating shareable links for broader collaboration.

Core Functionalities

  • Embed Ziflow's proofing interface directly into the project management platform via an iframe.
  • Enable the creation of proofs in Ziflow using REST API calls directly from the platform.
  • Allow users to add participants to proof review stages for improved collaboration and accountability.
  • Provide the ability to generate shareable links for proof access, facilitating external feedback and reviews.
  • Maintain Ziflow's core proofing functionalities, such as annotations, comments, and version tracking, ensuring a comprehensive review process.
  • Automate task creation based on proof review outcomes, improving project tracking and management.
